# GraphQL TodoMVC

This project is a demonstration of building realtime data driven application with GraphQL.


## Install

```bash
npm install
```


## Build & Run

Build the application:

```bash
npm run build
```

Start server:

```bash
npm start
```


## Develop

```bash
npm run dev
```
